This document describes the OpenTelemetry integration in {product_name}, providing comprehensive observability capabilities for workflow applications.

{product_name} integrates with OpenTelemetry to provide:

1. **Process Instance Aware Context**: Context isolation and enhanced observability across concurrent workflow executions
2. **OpenTelemetry Tracing**: Workflow execution tracing with proper context propagation
3. **Context Propagation**: Maintains proper context separation between different process instances during concurrent execution
